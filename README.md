## Installation

1. Clone this repo
```sh
git clone https://github.com/dev-rahul-0/stationery-apk
```
2. run this command to fetch the dependencies
```sh
flutter pub get
```
3. now build your APK using following command:
```sh
$ flutter build apk 
```
